isAsync

isAsync: boolean = false
  • Defined in index.ts:91

Indicates whether the pipeline includes asynchronous processing.

If true, items can only be pushed into the pipeline using the .pushAsync() method, otherwise, the pipeline is more efficiently invoked by the .push() method.
